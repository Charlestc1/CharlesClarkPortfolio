# Electric Vehicle Adoption Analysis in Washington State

This project explores the adoption and geographic distribution of electric vehicles (EVs) across Washington State using vehicle registration data from the Washington State Department of Licensing (DOL). The analysis focuses on Battery Electric Vehicles (BEVs) and Plug-in Hybrid Electric Vehicles (PHEVs), examining how adoption patterns vary across counties and cities and how the composition of EV types differs across regions.

Through exploratory data analysis and visualization, the project highlights areas leading EV adoption and evaluates trends in vehicle registrations and electric range over time by model year. Particular attention is given to how the balance between BEVs and PHEVs differs geographically, reflecting variations in infrastructure, demographics, and regional preferences.
